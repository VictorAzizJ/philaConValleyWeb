# Community Website

This project is all about creating a central hub for the Philadelphia tech community.

## Features

- **Content Management**: MDX-powered blog and project showcase
- **Dark Mode**: Full theme support for better accessibility
- **Performance**: Built with Next.js App Router for optimal speed
